Definition of Done (global)

  • Código revisado e merge na branch principal
  • Testes unitários/integração BE passando
  • Caso QA da história executado e documentado
  • Sem regressão nos módulos integrados adjacentes
  • RBAC validado para papéis Colaborador, Gestor, RH, Admin
  • Audit trail para operações sensíveis (movimentações, SSO)
  • Documentação API atualizada (se endpoint novo)
  • Maturidade atualizada em page-maturity.ts (se rota nova/alterada)
  • Mapa do site revisado se nova rota (sitemap.ts)
  • Demo script R1–R4 atualizado se entregável de release

Ferramentas

  • Vitest — testes unitários FE
  • Playwright — E2E (npm run test:e2e)
  • RM Homolog — Corpore SQL + workers TOTVS
  • Observabilidade — trilha auditoria admin
